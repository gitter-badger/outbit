Description
===========

outbit - The command line and control center of a Data Center or Cloud.


Installation
===========

```shell
pip install outbit
```

or

```shell
easy_install outbit
```

Usage
===========

### Usage

$ outbit
  Username: xxxxxx
  Password: xxxxxx

user@outbit> help
  webserver
  apps
  useraccounts
 
user@outbit> help webserver
  webserver restart  [all|group|host]
